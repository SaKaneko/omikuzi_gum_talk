# 要求
- ランダムでトークテーマを提示するWebアプリケーション
- 主にpythonで動作する
- docker-composeでアプリケーションを起動し、Webサーバーとして動作する
- 保守性を高めるために、オブジェクト指向で実装する。
- 日本語をキーとして複数の言語に対応する。複数言語の対応辞書は.po, .moファイルで実現する。
- 画面構成は以下の通り
    - メイン画面
        - おみくじ画面に遷移するか、話題一覧画面に遷移するか、話題提供画面に遷移するかを選択する。
    - おみくじ画面
        - おみくじアニメーションと共に、話題ディレクトリにあるファイルの中からランダムに選択し、おみくじ結果画面に遷移する
    - おみくじ結果画面
        - ランダムに選択された話題ファイルの内容をmd形式でしゅつりょくする。画面下部に話題を削除してメイン画面に戻るボタンと、メイン画面に戻るボタンを持つ
    - 話題を投稿する画面
        - 話題を投稿する。題名と中身の入力欄がある。入力欄は複数行の入力を受け付け、マークダウン形式で入力できる。previewボタンを押すと現在の入力状態でmarkdownとして解釈させたpreviewを表示し、投稿ボタンを押すと話題ディレクトリに回答内容を格納したファイルを出力する。
    - 話題一覧画面
        - 現在話題ディレクトリの中にあるファイルの先頭行にある題名を一覧で表示する。中身は見れない。削除ボタンで話題を削除することもできる。
